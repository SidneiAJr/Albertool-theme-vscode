# 🎨 Albertool Themes | Pack

Coleção de temas escuros para VS Code desenvolvidos pela **Albertool** — cada tema com identidade visual única, fundo preto e paletas cuidadosamente escolhidas para longas sessões de código.

---

## 🖌️ Temas Disponíveis

### ⬛ Albertool Simples
Minimalista. Preto puro com cinza suave — sem distrações, só código.

---

Preto com roxo neon, verde elétrico e azul vibrante. Visual cyberpunk limpo e moderno.

---

### 🔴 Albertool Communist
Vermelho profundo com dourado. Clássico, sombrio e marcante.

---

### 👑 Albertool Prime
Preto com dourado suave. Elegante e minimalista para quem prefere tons quentes.

## 📦 Como instalar

1. Abra o VS Code
2. Pressione `Ctrl+Shift+X`
3. Pesquise `Albertool Themes`
4. Clique em **Install**
5. Pressione `Ctrl+K Ctrl+T` e escolha o tema
